# AI STEP 1

## 이 STEP에서 왜 이걸 배우는가

AI는 나중에 붙이는 옵션처럼 보이지만, 실제로는 출력 형식을 처음부터 못 잡으면 가장 늦게 흔들리는 영역이다.  
모델 자체에 집중하기 쉬운데, 이 프로젝트에서는 "어떤 입력을 주고 어떤 형식으로 받을지"가 더 중요하다.

## 이번 프로젝트에서 어디에 쓰는가

- 액션 아이템 추출
- 톤 변환
- 진행 상황 요약
- 팀 문서 Q&A

## 반드시 알아야 하는 체크리스트

- [ ] AI 요청도 결국 입력과 출력 약속이 필요하다는 것
- [ ] JSON처럼 일정한 형식이 있어야 연동이 쉽다는 것
- [ ] 실패했을 때 어떻게 보일지도 같이 정해야 한다는 것

## 막히면 이렇게 생각하면 된다

- 모델 이름보다 출력 필드명이 먼저다.
- 한 번에 똑똑한 답을 받는 것보다 안정적인 형식이 더 중요하다.
- 실패했을 때 버튼을 숨길지, 문구를 보여 줄지 같이 적는다.
